# AngularAnimalPaintings

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.2.16.

## Development server[Amazing Animal Paintings.json](https://github.com/user-attachments/files/23583071/Amazing.Animal.Paintings.json)


Run `ng serve -o` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

**You do need to install mockoon on your device in order to see some functionality.**
Check out their documentation below to learn more:
https://mockoon.com/

Once you have it installed on your system, import the JSON file included here.
[Mockoon environment JSON](./Amazing%20Animal%20Paintings.json)

To get more help on the Angular CLI use `ng help` or go check out the [Angular CLI Overview and Command Reference](https://angular.io/cli) page.
